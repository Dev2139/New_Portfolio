# Dev Patel - Personal Portfolio Website

A modern, responsive portfolio website showcasing my skills, projects, certificates, and contact information. Featuring a sleek dark theme with orange accents.

## Features

- **Responsive Design**: Works seamlessly across desktop, tablet, and mobile devices
- **Interactive Navigation**: Smooth scrolling to different sections with hash anchors
- **Project Showcase**: Grid layout displaying projects with hover effects
- **Certificate Gallery**: Interactive certificate showcase with zoom functionality
- **Contact Form**: Functional contact form integrated with Formspree
- **Social Media Integration**: Links to all major social platforms
- **Modern UI Elements**: Hover animations, gradient effects, and smooth transitions

## Sections

### About
- Professional introduction and summary
- Downloadable resume link
- Services offered

### Background
- Education history
- Work experience
- Technical and design skills with progress bars

### Projects
- Grid layout showcasing projects
- Filtering capabilities
- Project details with links to live demos and repositories

### Certificates
- Interactive gallery of certificates and achievements
- Click to view full-size versions

### Contact
- Contact information
- Functional contact form
- Direct contact options
